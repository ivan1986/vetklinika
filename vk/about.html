Описание программы

«Ветеринарная клиника» - программа, преднозначенная для использования ветеринарными врачами небольших клиник или частно практикующими врачами.

Несмотря на свой маленький размер «Ветеринарная клиника» имеет удобный интерфейс, проста в обслуживании, позволяет автоматизировать функции по ведению журналов амбулаторного приема и вакцинации при работе с пациентами в ветеринарных клиниках.

Программа содержит две независимые части:

  • Журнал амбулаторного приема
  • Журнал вакцинации

Каждая из частей находится на отдельной вкладке.

На этих вкладках находятся: основная таблица записей о пациентах и дополнительные таблицы записей о каждом пациенте, которые содержат данные текущего пациента и меняются при выборе другого пациента.

Системные требования

Программа не требовательна к ресурсам, работает на любом компьютере с операционной системой Windows. Для работы в операционной системе Windows – 95 необходим компонент MDAC-2.1, который можно скачать с сайта Майкрософт.
